# Luma Launcher

A customizable, favorites-first launcher for Android TV and Google TV.

## Install on Google TV

1. Open the latest release on your TV using a browser or the Downloader app.
2. Download `Luma-Launcher-v0.1-debug.apk`.
3. If prompted, allow that browser or Downloader to install unknown apps.
4. Open the APK and choose **Install**.
5. Open Luma, select **Customize**, then **Choose Luma as default Home**.

## Current features

- Favorites-first TV layout
- Automatic local weather with manual city choices
- Network and music widgets
- Custom backgrounds and app-card designs
- Full replacement artwork for supported apps
- Manual **Change icon** option for any favorite
- Android TV remote navigation

## Important

This is an early debug build intended for personal testing. It is not a Play Store release.
